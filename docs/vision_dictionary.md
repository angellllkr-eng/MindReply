# MindReply Vision Dictionary

MindReply is an Executive Nervous System and Decision Infrastructure Layer. It sits between input and action, reducing hesitation before communication or follow-up happens.

## Category Terms

- Executive Nervous System
- Decision Infrastructure
- decision layer
- private decision filter
- intake layer
- action layer
- memory layer
- recommended action
- risk check
- quiet receipt
- professional playbook
- calm execution
- controlled access

## Microcopy

- The next move is obvious.
- Paste the pressure.
- One synthesis. One recommended action.
- Risk checked before movement.
- Memory adjusted quietly.
- Private by default.
- Hold and review.
- Set the follow-up.
- Send the prepared reply.
- Mark resolved.

## Interface Rules

- Show one input field.
- Show one synthesis.
- Show one recommended action.
- Show one risk status.
- Show one receipt.
- Keep memory quiet unless the user asks.
- Remove screens that create extra decisions.

## Feature Evaluation

Every feature must remove friction, simplify a decision, increase trust, and support the category. If it does not make the next move obvious, it does not belong in this pass.
